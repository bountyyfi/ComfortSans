# ComfortSans™

**"The font that cares about your eyes"**

A clean, comfortable sans-serif optimized for screen readability. Perfect for web apps, dashboards, and design mockups.

## Quick Start

### Web (CSS)

```css
@font-face {
  font-family: 'ComfortSans';
  src: url('./ComfortSans-Regular.woff2') format('woff2'),
       url('./ComfortSans-Regular.ttf') format('truetype');
  font-weight: normal;
  font-style: normal;
}

body {
  font-family: 'ComfortSans', sans-serif;
}
```

### Desktop

Double-click `ComfortSans-Regular.ttf` → Install. Available in Photoshop, Illustrator, Figma (desktop), Word, and all other applications.

### Figma / Canva

Upload the `.ttf` file as a custom font. Works immediately across all your designs.

## Features

- **Geometric block letterforms** — clean, modern, no-nonsense
- **Optimized for screens** — crisp at any size
- **Full character set** — A-Z, a-z, 0-9, punctuation
- **OpenType features** — contextual alternates (`calt`) for improved readability
- **Tiny footprint** — 1.5KB WOFF2, loads instantly
- **SIL Open Font License** — free for personal and commercial use

## OpenType Features

ComfortSans includes `calt` (Contextual Alternates) which are **enabled by default** in all browsers and most desktop applications. These provide optimized glyph rendering for common character sequences, improving overall readability.

## Installation

### macOS
Double-click the `.ttf` file → Click "Install Font"

### Windows
Right-click the `.ttf` file → "Install" or "Install for all users"

### Linux
Copy to `~/.local/share/fonts/` then run `fc-cache -fv`
